# Паттерны «Банды четырёх (GOF)»
Папка Creational содержит порождающие паттерны:
- Абстрактная фабрика
- Строитель
- Фабричный метод
- Прототип
- Одиночка

Папка Structural содержит структурные паттерны:
- Адаптер
- Мост
- Компоновщик
- Декоратор
- Фасад
- Приспособленец
- Заместитель

Папка Behavioral содержит поведенческие паттерны:
- Цепочка обязанностей
- Команда
- Интерпретатор
